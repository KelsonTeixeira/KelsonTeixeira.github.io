Para otimizar um site:

  1. Olhar qual é o maior resolução que a imagem aparece no site e resuzir o tamanho da imagem original para o mesmo tamanho.
  2. Diminuir o tamanho em bytes das imagens (comprimir): optimizilla.com
  3. Minimizar o tamanho o css: csscompressor.com/
  4. Incluir uma descrição rápida e sucinta do site para aparecer no google (meta name="description" content="This is a anotation site")
  5. Validar o código HTML: validator.w3.org/check
  6. Usar o google analytics
  7. Melhorar a acessibilidade.
  8. No wordpress: Usar plugin de cache
  9. Utilizar a extensão Lighthouse para avaliar o site: lighthouse
